  YouTube 汉化工具 - 说明书（人话版）
══════════════════════════════════════════════════════

这个工具是干嘛的？
────────────────
把英文的 YouTube 视频变成中文版。
- 原来是英文配音 → 变成中文配音
- 背景音乐保留
- 自动生成中文字幕


怎么用？
────────────────
1. 双击桌面的「YT汉化工具」图标
2. 把 YouTube 链接粘贴进去
3. 选一下想要男声还是女声
4. 点「汉化视频」
5. 等它跑完（会弹出一个黑色窗口显示进度）
6. 完成后视频在：D:\YouTube汉化工具\downloads\videos\


四个模式是什么意思？
────────────────
① 配音+背景音（默认）
   → 中文语音替换英文，保留背景音乐
   → 最推荐，效果最好

② 仅字幕嵌入
   → 不配音，只把中文字幕烧到视频画面上
   → 适合想保留原声的人

③ 仅生成音频
   → 只生成中文音频文件（MP3）
   → 不改视频，自己后期用

④ 仅字幕文件
   → 只生成字幕文件（SRT）
   → 用其他剪辑软件自己加字幕


四种声音有什么区别？
────────────────
男声  → 比较自然的男声（默认）
女声  → 比较自然的女声
男声2 → 另一种男声风格
女声2 → 另一种女声风格

建议都试试，选一个听着最舒服的。


文件保存在哪里？
────────────────
D:\YouTube汉化工具\downloads\videos\

打开工具后点「打开目录」按钮可以直接打开这个文件夹。

文件名说明：
  xxx.mp4     → 原版视频
  xxx.zh.mp4  → 汉化后的视频（用这个）


下载的视频太大怎么办？
────────────────
1080p 的 22 分钟视频大约 200MB，这是正常的。
如果嫌大，可以用「仅字幕嵌入」模式，不会增加太多体积。


汉化后的视频能发到抖音/B站吗？
────────────────
可以。生成的 MP4 文件是通用格式，任何平台都能上传。


工具用不了怎么办？
────────────────
按这个顺序排查：

1. 双击没反应
   → 试试双击 D:\YouTube汉化工具\start.bat

2. 提示 Python 未安装
   → 点「是」让它自动装，需要联网

3. 下载失败
   → 点「更新内核」按钮
   → 等几分钟再试（YouTube 限流）

4. 汉化后没声音
   → 检查 D:\YouTube汉化工具\bin\ffmpeg.exe 是否存在

5. 翻译不准确
   → 找有中文字幕的视频效果最好
   → 工具会优先用 YouTube 自带的中文字幕


怎么更新工具？
────────────────
打开工具 → 点「更新内核」按钮

会自动检查新版本，失败了会自动恢复，不用担心弄坏。


怎么分享给别人？
────────────────
双击 D:\YouTube汉化工具\pack.bat
→ 会生成一个压缩包
→ 把压缩包发给朋友
→ 朋友解压后双击 start.bat 就能用
→ 首次使用会自动安装 Python（需要联网）


怎么彻底卸载？
────────────────
1. 删除桌面上的「YT汉化工具」图标
2. 删除 D:\YouTube汉化工具 整个文件夹
3. 完事


有什么限制？
────────────────
1. 需要联网（下载视频、翻译、语音合成都需要）
2. 不是所有视频都能下载（付费/Private 不行）
3. 中文语音听起来像真人但还是能听出是 AI
4. YouTube 限流时需要等几分钟重试
5. 翻译质量取决于 YouTube 自带字幕，没有中文字幕的视频翻译会差一些


文件夹里都有什么？（能不能删）
────────────────
D:\YouTube汉化工具\
│
├── bin\              ← 引擎文件（不能删！删了工具就废了）
│   ├── yt-dlp.exe   ← 负责从 YouTube 下载
│   ├── ffmpeg.exe    ← 负责处理视频音频
│   ├── ffprobe.exe   ← 负责读取视频信息
│   └── dubber.py     ← 负责汉化的核心代码
│
├── downloads\        ← 你下载的内容（可以删，就是你的视频）
│
├── launcher.ps1      ← 工具的界面（不能删）
├── start.bat         ← 启动器（不能删）
├── start.vbs         ← 另一个启动器（不能删）
│
├── pack.bat          ← 打包脚本（可以删，不影响使用）
├── make_shortcut.vbs ← 建快捷方式脚本（可以删）
├── readme.txt        ← 说明（可以删）
├── 快速参考.txt      ← 速查卡（可以删）
├── 维护手册.md       ← 维护文档（可以删）
├── 技术架构.md       ← 技术文档（可以删）
├── AI接手指南.md     ← 给AI看的（可以删）
└── 人话版说明.txt    ← 就是这个文件（可以删）


遇到解决不了的问题？
────────────────
1. 先看这个文件有没有写
2. 点「更新内核」试试
3. 关掉重新打开试试
4. 重启电脑试试
5. 找 AI 帮忙（把 AI接手指南.md 发给它）
