# YouTube 汉化工具 - 维护手册

## 目录结构

```
D:\YouTube汉化工具\
├── bin\                    ← 核心引擎（不要删除）
│   ├── yt-dlp.exe         ← YouTube 下载引擎
│   ├── ffmpeg.exe          ← 音视频处理
│   ├── ffprobe.exe         ← 媒体信息分析
│   └── dubber.py           ← 汉化核心脚本
├── downloads\              ← 所有下载内容
│   ├── videos\             ← 视频（原版 + .zh.mp4 汉化版）
│   ├── audio\              ← 音频（用「下载音频」时生成）
│   └── subtitles\          ← 字幕文件
├── temp\                   ← 临时文件（自动清理）
├── launcher.ps1            ← GUI 主程序
├── start.bat               ← 启动器（双击）
├── start.vbs               ← 无窗口启动器
├── pack.bat                ← 打包分享
├── make_shortcut.vbs       ← 重建桌面快捷方式
└── readme.txt              ← 使用说明
```

## 日常维护

### 1. 更新 yt-dlp（最重要）

**频率**：每月 1 次，或下载失败时

**方法**：
- 打开工具 → 点「更新内核」
- 会自动备份，失败可恢复

**为什么需要更新**：
- YouTube 经常改接口，旧版 yt-dlp 会下载失败
- 新版修复 bug、支持更多格式

**手动更新**：
```bash
D:\YouTube汉化工具\bin\yt-dlp.exe -U
```

### 2. 清理下载文件

**位置**：`D:\YouTube汉化工具\downloads\`

**建议**：
- 定期删除不需要的视频（1080p 22分钟 ≈ 200MB）
- `.zh.mp4` 是汉化版，`.mp4` 是原版
- 确认汉化版满意后再删原版

### 3. 更新 Python 包（偶尔）

**何时**：翻译或语音功能报错时

**方法**：
```bash
# 打开命令提示符（Win+R 输入 cmd）
pip install --upgrade edge-tts deep-translator
```

## 常见问题及解决

### 下载类

| 问题 | 原因 | 解决 |
|------|------|------|
| 下载失败，提示 403/429 | YouTube 限流 | 等 5-10 分钟重试 |
| 下载失败，提示版本过旧 | yt-dlp 版本太老 | 点「更新内核」 |
| 下载速度很慢 | 网络问题或 YouTube 限速 | 换时间段重试 |
| 提示 "No JavaScript runtime" | 警告，不影响使用 | 忽略即可 |

### 字幕类

| 问题 | 原因 | 解决 |
|------|------|------|
| 没有中文字幕 | 视频本身没有中文 | 工具会自动用英文翻译 |
| 字幕翻译质量差 | 机器翻译不准确 | 找有中文字幕的视频 |
| 字幕下载 429 错误 | YouTube 限流 | 等几分钟重试 |
| 字幕和语音对不上 | 时间轴偏移 | 正常现象，逐句对齐会有微小偏差 |

### 语音类

| 问题 | 原因 | 解决 |
|------|------|------|
| 语音生成失败 | Edge TTS 服务波动 | 重新运行即可 |
| 语音听起来机械 | TTS 本身的限制 | 换「女声」试试 |
| 语音和画面不同步 | 语速调整偏差 | 选「仅字幕嵌入」模式 |
| 部分句子没声音 | 特殊字符导致 TTS 失败 | 正常，跳过即可 |

### 视频类

| 问题 | 原因 | 解决 |
|------|------|------|
| 汉化后没声音 | ffmpeg 异常 | 检查 bin\ffmpeg.exe 存在 |
| 视频播放需要新编解码器 | 编码格式问题 | 已用 H.264，不应出现 |
| 汉化文件很大 | 1080p 视频本身大 | 正常，22分钟约 200MB |
| 汉化文件很小 | 合成失败 | 重新运行 |

### 软件类

| 问题 | 原因 | 解决 |
|------|------|------|
| 双击没反应 | VBS 启动器问题 | 双击 start.bat 试试 |
| Python 未安装 | 系统重装或未安装 | 工具会自动提示安装 |
| 界面乱码 | 编码问题 | 重新下载工具包 |
| 更新内核闪退 | 旧版本 bug | 用新版 launcher.ps1 |

## 高级操作

### 手动运行汉化（命令行）

```bash
# 基本用法
python D:\YouTube汉化工具\bin\dubber.py "YouTube链接"

# 指定语音和模式
python D:\YouTube汉化工具\bin\dubber.py "链接" --voice 女声 --mode dub

# 只生成字幕
python D:\YouTube汉化工具\bin\dubber.py "链接" --mode srt

# 只生成音频
python D:\YouTube汉化工具\bin\dubber.py "链接" --mode audio

# 字幕嵌入视频
python D:\YouTube汉化工具\bin\dubber.py "链接" --mode subtitle
```

### 可用语音

| 名称 | 声音 |
|------|------|
| 男声 | Yunxi（默认，自然男声） |
| 女声 | Xiaoxiao（自然女声） |
| 男声2 | Yunjian（另一男声） |
| 女声2 | Xiaoyi（另一女声） |

### 重建桌面快捷方式

如果桌面快捷方式丢失：
```bash
cscript D:\YouTube汉化工具\make_shortcut.vbs
```

### 打包分享给别人

```bash
# 运行打包脚本
D:\YouTube汉化工具\pack.bat

# 生成的 zip 文件
D:\YouTube汉化工具\YouTube汉化工具.zip
```

**对方使用**：
1. 解压 zip
2. 双击 `start.bat`
3. 首次会自动安装 Python（需联网）
