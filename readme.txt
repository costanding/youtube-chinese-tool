  YouTube 汉化工具 - 使用说明
══════════════════════════════════════════

【首次使用】
1. 双击"启动.bat"
2. 首次会自动安装 Python（需联网，约 30MB）
3. 自动安装翻译和语音依赖包
4. 等待界面出现

【功能说明】
┌─────────────┬────────────────────────┐
│ 汉化视频    │ 下载+翻译+中文配音     │
│ 下载视频    │ 最高画质 H.264 MP4     │
│ 下载音频    │ 提取为 MP3             │
│ 下载字幕    │ 英/中/日字幕           │
│ 视频信息    │ 查看标题、时长等       │
└─────────────┴────────────────────────┘

【汉化模式】
- 配音+背景音：中文语音 + 原始背景音乐
- 仅字幕嵌入：中文字幕烧录到视频画面
- 仅生成音频：只生成中文音频文件
- 仅字幕文件：只生成 SRT 字幕

【语音选择】
- 男声：自然男声（默认）
- 女声：自然女声
- 男声2/女声2：备选声音

【文件保存位置】
downloads\videos\  - 视频文件
downloads\audio\   - 音频文件
downloads\subtitles\ - 字幕文件

【分享给朋友】
双击"打包分享.bat"，会生成一个 zip 文件
把 zip 发给朋友，对方解压后双击"启动.bat"即可

【更新引擎】
点击"更新内核"按钮即可更新 yt-dlp

【常见问题】
Q: 提示 Python 未安装？
A: 点"是"自动安装，或手动安装 Python 3.10+

Q: 翻译失败？
A: 工具会自动重试 3 次，并使用备用翻译引擎

Q: 字幕下载失败？
A: YouTube 限流，等几分钟重试

Q: 汉化后音画不同步？
A: 工具会自动调整语速对齐，如有偏差可选"仅字幕嵌入"模式
